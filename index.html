Getting started


Add sceletus.css to your page

You can use sceletus.css on your page by adding the following <link> element into your page's <head>, before your project's stylesheets.


CODE EXAMPLE 
<link href="path/to/sceletus.css" rel="stylesheet"/>

Alternatively you can download sceletus.css here


Add the Viewport Meta Element

The viewport meta element lets you control the the width and scale of the viewport on mobile browsers. Since you're building a responsive website, you want the width to be equal to the device's native width. Add this into your page's <head>.


CODE EXAMPLE 
<meta name="viewport" content="width=device-width, initial-scale=1">

Typography

The typography base is Open Sans served by Google, set at 15px over a 1.6 line height (24px). Other types like anchors, strong, emphasis, and underline are all included.


Headings come in six distinct sizes each with specific letter-spacing, line-height, and margins.

Bolded, Italicized, Colored, Underlined

Buttons

To make a button, simply add the .button class to an element followed by a color and a size.

The buttons come in 6 different colors: .green .blue .red .yellow .light .dark They also come in 3 different sizes. Add .big to make big button and .small to make a small one. To make a normal button, don't add a size class.


Regular buttons

Green button Blue button Red button Yellow button Light button Dark button

Big buttons

Green button Blue button Red button Yellow button Light button Dark button

Small buttons

Green button Blue button Red button Yellow button Light button Dark button


CODE EXAMPLE 
<!-- Big red button -->
<button class="button red big"></button>

<!-- Regular green button -->
<a class="button green" href="#">Green button</a>

<!-- Small yellow button -->
<input type="submit" class="button yellow small" />

Lists

  • Unordered lists have basic styles
  • They use the circle list style
    • Nested lists use the disc list style
    • Unordered lists can nest either type of list
  • Just another list item
  1. Ordered lists have basic styles
  2. They use the decimal list style
    • Ordered lists can nest either type of list
    • Nested lists have a smaller font-size
  3. Another list item

<ol>

   <li>Oredere list item 2

Forms

Sceletus.css aims to reinvent the input layout model, thus sceletus.css has adopted a very simplistic layout.
The colours can be easily changed in the.scss file.


To use the sceletus form styles, you must wrap the inputs with a .sceletus_form class. You can apply the class on a div element, or directly on the form.



Login form

CODE EXAMPLE 
<form class="sceletus-form">
   <input type="email" placeholder="Email">
   <input type="password" placeholder="Password">
   <input class="button blue" type="submit" value="Sign In">
   <input class="pull-right" id="remember" type="checkbox">
   <label class="pull-right" for="remember"> Remember me</label>
</form>

Contact form

CODE EXAMPLE 
<form class="sceletus-form">
   <input type="text" placeholder="Name">
   <input type="email" placeholder="Email">
   <textarea placeholder="Message"></textarea>
   <input class="button blue full-width" type="submit" value="Send">
</form>

Alert Boxes

Sceletus comes with a few alert boxes. By default they will fill the width of the container they're placed in.

There are currently 4 different alerts: .message, .success, .warning and .error.


By continuing to use this site, you agree to our cookie policy
Success! Your settings were changed
Warning! Your post has been deleted!
Whoops, something went horribly wrong!

CODE EXAMPLE 
<!-- (Blue) Message alert -->
<input id="message" type="checkbox" hidden="hidden" name="message" tab-index="0">
<div class="sceletus-alert message">
    By continuing to use this site, you agree to <a href="#0">our cookie policy</a>
    <label for="message" class="close">&#10006</label>
</div>

<!-- (Green) Success alert -->
<input id="success" type="checkbox" hidden="hidden" name="success" tab-index="0">
<div class="sceletus-alert success">
    Success! Your settings were changed
    <label for="success" class="close">&#10006</label>
</div>

<!-- (Yellow) Warning alert -->
<input id="warning" type="checkbox" hidden="hidden" name="warning" tab-index="0">
<div class="sceletus-alert warning">
    Warning! Your post has been deleted!
    <label for="warning" class="close">&#10006</label>
</div>

<!-- (Red) Error alert -->
<input id="error" type="checkbox" hidden="hidden" name="error" tab-index="0">
<div class="sceletus-alert error">
    Whoops, something went horribly wrong!
    <label for="error" class="close">&#10006</label>
</div>

Responsive Videos

To use the sceletus responsive video, simply wrap your video iframe in a container with class sceletus-video. Sceletus presumes that the aspect ratio of the video is 16:9 and scales accordingly. Remember to remove the width and height property from the iframe tag.



CODE EXAMPLE 
<div class="sceletus-video">
  <iframe src="https://www.youtube.com/embed/Dm0YnuQeROI"></iframe>
</div>

Image shapes

Images in sceletus can be made responsive by using the .sceletus-img class.
This applies max-width: 100%;, height: auto; and display: block; to the image so it scales to the parent element. See the Utilities section for more details



Rounded corners

Add the .rounded class to your image


Circle

Add the .circle class to your image


Border

Add the .border class to your image



CODE EXAMPLE 
<!--Rounded corners-->
<img src="http://lorempixel.com/800/800" class="sceletus-img rounded" alt="Rounded corners" />

<!--Circle-->
<img src="http://lorempixel.com/800/800" class="sceletus-img circle" alt="Circle" />

<!-- Border-->
<img src="http://lorempixel.com/800/800" class="sceletus-img border" alt="Border" />

Queries

sceletus.css uses media queries to generate its scalable grid. It comes with a list a queries that makes it easy to style you projects across devices. The queries are mobile first, which means they target "min-width".

  • Mobile: 768px
  • Tablet: 992px
  • Desktop: 1200px

CODE EXAMPLE 
/* Larger than mobile */
@media (min-width: 768px) { }

/* Larger than tablet */
@media (min-width: 992px) { }

/* Larger then desktop */
@media (min-width: 1200px) { }
